# OIBSIP_GraphicDesign_Task2
🦋 Task 2: Logo Design – "Modest Haya"

Name: Abilashiny Selvakumaran
Batch: February-P1 2026
Domain: Graphic Design Intern

🎯 Objective

Create a minimal, elegant, and feminine logo that represents modesty, beauty, and grace for the fictional brand “Modest Haya”.

🛠 Steps Performed

Conceptualization: Researched minimalist and feminine logo styles.

Sketching: Created rough sketches of logo ideas combining a heart-shaped line and a butterfly element.

Digital Design: Used Canva to digitize the preferred concept.

Typography & Layout: Selected clean, elegant fonts and arranged the logo to ensure balance and memorability.

Mockup Presentation: Placed the logo on a brand mockup to visualize usage on products.

Final Export: Exported logo as PNG and PDF for sharing and documentation.

🖌 Tools Used

Canva

✅ Outcome / Result

A feminine and elegant logo that communicates modesty, beauty, and self-care.

Balanced composition with clean typography.

Ready for branding applications and portfolio display.
